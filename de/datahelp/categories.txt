<p>
	Hier werden Ihre internen Kategorien inkl. der Ebenen-Struktur aufgelistet. Sie können bereits bestehende Kategorien
	bearbeiten und exportieren oder neue Kategorien und Unterkategorien anlegen bzw. importieren.
</p>
 
<ul>
	<li><h4>Interne Kategorien</h4>
		Interne Kategorien sind alle <strong>Kategorien im Datenpool</strong>, in denen Ihre Produkte bzw. Datensätze 'intern' zugeordnet sind.
		Diese Kategorien werden beim Import von Datensätzen im Datenpool angelegt. Enthält Ihre Importdatei bereits Kategorien,
		so können diese als interne Kategorien verwendet werden. Falls kein Kategorien in Ihrer Importdatei enthalten sind,
		vergibt unser System eigene Kategorie-Nummern. Über die internen Kategorien erfolgt später im Bereich <i>Schnittstellen</i>
		die Zuordnung zu den Kategorien Ihrer CSV- oder XML-Schnittstelle.<br />
		Um eine Kategorie zu <strong>bearbeiten</strong>, klicken Sie auf den Namen der Kategorie. Es öffnet sich eine neue Seite,
		auf der Sie die Eigenschaften und Werte der entsprechenden Kategorie einstellen können.<br />
		<br />
		<i>Nachdem Sie eine einzelne Kategorie angeklickt haben...</i><br />
		<strong>[ Einstellungen ] Kategorie-ID, Parent-ID, Name, Spracheinstellungen</strong> - Ändern Sie hier die Einstellungen
		der internen Kategorie.<br />
		<strong>[ eBay ] Kategorie-ID für "..."</strong> - Tragen Sie hier eine korrekte Bay-Kategorie ein oder klicken Sie auf [ Suche ],
		um eine passende eBay-Kategorie auszuwählen. Über den Button [ ID übernehmen ] wird die Kategorie in das jeweilige Feld eingetragen.
		Speichern Sie Ihre hinzugefügten oder geänderten Inhalte.
	</li>
	<li><h4>Weitere Aktionen</h4>
		<strong>Alle Kategorien zum Löschen markieren...</strong> - Hier haben Sie die Möglichkeit, ALLE internen Kategorien zu löschen.
		Über den Button <span class="txtRed">[ Löschen ]</span> werden alle internen Kategoriezuordnungen gelöscht. Produktdatensätze
		werden dabei nicht gelöscht. Nach dem Löschen können Sie Ihre internen Kategorien über den Import wieder neu anlegen bzw.
		Ihren Datensätzen im datenpool neu zuordnen.<br />
        <br />
        Achtung! Artikelzuordungen werden beim Löschen von Kategorien entfernt. Arbeiten Sie mit mehreren Kategorie-Ebenen,
        werden zuerst immer die untersten Ebenen gelöscht.
	</li>
</ul>
