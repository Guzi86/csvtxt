<p>
	Hier werden Ihre internen Kategorien inkl. der Ebenen-Struktur aufgelistet. Sie können bereits bestehende Kategorien
	bearbeiten und exportieren oder neue Kategorien und Unterkategorien anlegen bzw. importieren.
</p>
 
<ul>
	<li><h4>Interne Kategorien</h4>
		Interne Kategorien sind alle <strong>Kategorien im Datenpool</strong>, in denen Ihre Produkte bzw. Datensätze 'intern' zugeordnet sind.
		Diese Kategorien werden beim Import von Datensätzen im Datenpool angelegt (oder manuell). Enthält Ihre Importdatei bereits Kategorien,
		so können diese als interne Kategorien verwendet werden. <span class="txtRed">Wichtig:</span> Markieren Sie dazu im Datenpool
		unter Vorlagen ->Import-Optionen die Option <i>"Kategorien aus Importdatei übernehmen bzw. abgleichen"</i>.<br />
		Sind kein Kategorien in Ihrer Importdatei enthalten, vergibt unser System eigene Kategorie-Nummern.
		Über die internen Kategorien erfolgt später im Bereich <i>Schnittstellen</i>
		die Zuordnung zu den Kategorien Ihrer CSV- oder XML-Schnittstelle.<br />
		Um eine interne Kategorie zu <strong>bearbeiten</strong>, klicken Sie auf den Namen der Kategorie. Es öffnet sich eine neue Seite,
		auf der Sie die Eigenschaften und Werte der entsprechenden Kategorie einstellen können.<br />
		<br />
		<i>Nachdem Sie eine einzelne Kategorie angeklickt haben...</i><br />
		<strong>[ Einstellungen ] Kategorie-ID, Parent-ID, Name, Spracheinstellungen</strong> - Ändern Sie hier die Einstellungen
		der internen Kategorie.<br />
		<strong>[ Extern ] eBay CatID</strong> - Tragen Sie hier eine korrekte eBay-Kategorie ein oder klicken Sie auf [ Suche ],
		um eine passende eBay-Kategorie auszuwählen. Über den Button [ ID übernehmen ] wird die Kategorie in das jeweilige Feld eingetragen.
		Speichern Sie Ihre hinzugefügten oder geänderten Inhalte.<br />
		(weitere Anbindungen, z.B. zu Amazon, sind in Vorbereitung...)
	</li>
	<li><h4>Button <span class="txtRed">[ Alle Kategorien löschen ... ]</span></h4>
		Sie werden auf eine neue Seite weitergeleitet mit zusätzlicher Bestätigungsabfrage für die Löschaktion.
		Wenn Sie dort die Option ausführen, werden ALLE (internen) Kategorien innerhalb des aktiven Datenpools gelöscht.<br />
		Möchten Sie nur bestimmte Kategorien löschen, markieren Sie bitte nur die jeweiligen Optionsfelder links.<br />
		Nach dem Löschen können Sie die internen Produktkategorien über den Import wieder neu anlegen bzw. Ihren Datensätzen
		im Datenpool neu zuordnen.<br />
        <br />
		<span class="txtRed">Wichtig:</span> Produktdaten werden beim Löschen von Kategorien nicht gelöscht! Allerdings werden die Zuordungen von Produktdaten
		zu den gelöschten Kategorien entfernt.
	</li>
</ul>
