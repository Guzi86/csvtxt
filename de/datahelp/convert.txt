<p>
	Mit dem Konverter können Sie Ihre CSV-Daten in fertig vorgegebene CSV-Formate für JTL-Wawi, Rakuten, 1&1-Shop, Yatego u.a. Systeme umwandeln.
</p>
<ul>
	<li><h4>Datendatei konvertieren (max. 30MB)</h4>
		<strong>Variante A/B</strong> -  Dateien können entweder vom lokalen Rechner (A) oder per Link aus dem Internet ((B) eingelesen werden.<br />
		<strong>XML-Daten</strong> - Bitte markieren, wenn es sich um eine XML-Datei handelt.<br />
		<strong>Vorlage</strong> - Falls Sie bereits Vorlagen angelegt haben, können Sie hier eine auswählen.<br />
		Nachdem Ihre Datei vom System eingelesen wurde, erscheint ein neues Fenster...
	</li>
	<li><h4>Datei wurde geladen</h4>
		Weisen Sie im oberen Bereich die Eigenschaften für die zu konvertierende Datei zu. Es wird Ihnen dort auch der Dateiname
		und die Anzahl der temporär eingelesenen Datensätze angezeigt.<br />
		<br />
		<strong>Spaltentrenner:</strong> Spaltentrenner festlegen<br />
		<strong>Texterkennungszeichen:</strong> Texterkennungszeichen eingeben (falls verwendet)<br />
		<strong>Spaltennamen in Zeile:</strong> Hier die Zeile angeben, in der die Spaltennamen stehen (ist nicht immer die 1. Zeile)<br />
		<strong>Umwandeln in Datei für:</strong> In welches Datei-Format sollen die Daten umgewandelt werden?<br />
		<strong>Vorlage Spaltenzuordnung:</strong> Bitte beachten Sie, dass eine gewählte Vorlage zum Dateiformat passen muss,
		in das Sie Ihre Daten konvertieren möchten.<br />
		<br />
		Drücken Sie dann auf den Button [ Aktualisieren ] für eine aktuelle Ansicht im Bereich 'Spalten zuweisen'.
	</li>
	<li><h4>Spalten zuweisen</h4>
		Ordnen Sie hier nun die Werte aus Ihrer Datei den jeweiligen Spalten der Datei zu, die Sie oben ausgewählt haben
		bzw. in die Ihre Daten konvertiert werden sollen.
	</li>
	<li><h4>Vorlage speichern</h4>
		Optional lässt sich die Zuordung hier als neue Vorlage speichern. Bitte wählen Sie dazu einen aussagekräftigen Namen - z.B.
		'KONVERTER-Vorlage Yatego'.
	</li>
</ul>
